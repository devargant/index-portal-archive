# index-portal-archive

## 项目简介

`index-portal-archive` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定的域名或网站，仅作为一组静态页面的集中存储与展示空间。每个页面均为独立文件，可单独访问或部署于任意静态托管服务。

## 目录结构

```
index-portal-archive/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── index.html      # 可选的入口页面，提供页面列表或导航
└── README.md       # 本文件
```

- `pages/`：每个独立 HTML 页面以单独文件形式存放，命名清晰，便于查找。
- `assets/`：页面共享或专用的资源文件，建议按页面或功能分类存放。
- `index.html`（可选）：可作为仓库首页，提供已归档页面的索引或简要说明。

## 页面归档说明

- 所有页面均为纯前端 HTML，不依赖后端服务。
- 页面内容保持原始结构，不进行额外修改或包装。
- 归档目的仅为保存与展示，不涉及任何推广、商业化或导向行为。
- 如需新增页面，请将文件放入 `pages/` 目录，并在必要时更新 `index.html` 中的链接。
